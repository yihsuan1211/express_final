# Express 基礎驗收

## Features - 產品功能  
- 簡易的網頁應用頁面，共有四個頁面
- 點擊四個不同的按鈕，會呈現相對應的文字頁面

## Environment SetUp - 環境建置
Node.js

## Installation - 安裝
1. 打開終端機，clone專案到本機  
```
git clone https://github.com/yihsuan1211/express_final.git
```

2. 進入專案資料夾  
```
cd express_final
```

3. 載入相關套件，終端機輸入  
```
npm install
```

4. 啟動伺服器，終端機輸入  
```
npm run dev
```

5. 順利運行  
  看見”listening on localhost:3000“代表順利運行，即可點擊網址：http://localhost:3000

6. 暫停運行  
```
ctrl + c
```
